# 📤 Экспорт данных для внешнего ИИ-анализа

Эта папка содержит всё необходимое для анализа твоих данных через внешние ИИ-сервисы (ChatGPT, Claude, Gemini и др.).

## 📁 Файлы в этой папке

### 1. `master-prompt.txt`
Готовый промт для ИИ-анализа. Содержит:
- Инструкции для ИИ о том, как анализировать данные
- Структуру ожидаемого ответа (7 разделов)
- Рекомендации по стилю анализа

**Этот файл уже готов к использованию!** Просто скопируй его содержимое.

### 2. `my-data.txt` (создаётся автоматически)
Твои личные данные из приложения:
- Профиль "Я сейчас"
- Профиль "Кем хочу стать"
- Все ежедневные чекины

Этот файл создаётся, когда ты нажимаешь кнопку **"📤 Экспорт для ИИ"** в приложении.

## 🚀 Как использовать

### Шаг 1: Экспортируй данные
1. Открой приложение Quantum Leap
2. Нажми кнопку **"📤 Экспорт для ИИ"** в шапке сайта
3. Файл `my-data.txt` скачается автоматически
4. Сохрани его в эту папку (`export-for-ai`)

### Шаг 2: Подготовь промт
1. Открой файл `master-prompt.txt`
2. Скопируй всё его содержимое (Ctrl+A, Ctrl+C)

### Шаг 3: Получи анализ
1. Открой любой ИИ-сервис:
   - **ChatGPT**: https://chat.openai.com
   - **Claude**: https://claude.ai
   - **Gemini**: https://gemini.google.com
   - Или любой другой

2. Вставь мастер-промт в чат

3. Открой файл `my-data.txt`

4. Скопируй всё его содержимое

5. Вставь данные в тот же чат (после промта)

6. Получи детальный анализ своего месяца! 🎉

## 💡 Советы

### Выбор ИИ-модели

**Для лучшего качества анализа:**
- **ChatGPT-4** — отличный баланс качества и скорости
- **Claude Opus** — самый глубокий и эмпатичный анализ
- **Gemini Pro** — хорошо работает с большими объёмами данных

**Для быстрого анализа:**
- **ChatGPT-3.5** — быстро, но менее детально
- **Claude Haiku** — быстрый и дешёвый вариант

### Дополнительные вопросы

После получения основной сводки можешь задать уточняющие вопросы:
- "Какие конкретные действия помогут мне преодолеть [конкретный паттерн]?"
- "Проанализируй связь между моим настроением и [конкретными событиями]"
- "Какие книги или ресурсы могут помочь с [конкретной проблемой]?"

### Приватность

⚠️ **Важно:** Когда ты отправляешь данные в ChatGPT, Claude или другие сервисы, они обрабатываются на серверах этих компаний. Если для тебя критична приватность:
- Используй локальный ИИ через Ollama (встроенная функция в приложении)
- Или удали из `my-data.txt` особо личную информацию перед отправкой

## 🔄 Регулярное использование

Рекомендуется экспортировать данные и получать анализ:
- **В конце каждого месяца** — для отслеживания прогресса
- **Перед важными решениями** — для понимания текущего состояния
- **При ощущении застоя** — для выявления скрытых паттернов
